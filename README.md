# 🕯️ CandleTime

A minimalist focus timer app that simulates a burning candle to help you stay concentrated during study or work sessions.

---

## 🚀 Overview

CandleTime is an Android application built with Kotlin and Jetpack Compose. It transforms the traditional timer concept into a visual experience: you light a virtual candle and stay focused until it burns out.

The app is inspired by the idea of "study until the candle finishes", providing a simple but effective way to maintain deep focus.

---

## ✨ Features

* ⏱️ Preset and custom study durations
* 🕯️ Real-time candle burning animation
* 🔥 Visual feedback when session completes (candle extinguishes)
* 📊 Session history tracking
* 💾 Local data persistence using Room
* 🎵 Background sounds (optional, if implemented)

---

## 🏗️ Tech Stack

* Language: Kotlin
* UI: Jetpack Compose
* Architecture: MVVM
* State Management: ViewModel
* Database: Room

---

## 📱 How It Works

1. Choose a preset time or create a custom duration
2. Tap the "Light Candle" button
3. The candle starts burning
4. Stay focused until the candle extinguishes
5. Session is automatically saved to history

---

## 🧠 Motivation

Most timer apps feel mechanical. CandleTime introduces a visual metaphor that encourages users to stay focused naturally—just like watching a real candle burn while studying.
